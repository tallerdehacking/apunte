Ejercicios Apps Web

Web I#

  1. Crearse una cuenta en Hack The Box.

  2. Hacer el challenge Canvas de la categoría Misc en HTB.

  3. Resolver los labs de Information disclosure de PortSwigger. Necesitarán una cuenta para ver los labs de PortSwigger.

Web II#

  1. Completar el XSS Game hecho por Google. En este verán distintos tipos de XSS y tendrán que evadir filtros.

  2. Resolver los laboratorios de Cross-Site Scripting de PortSwigger.

  3. Resolver los laboratorios de SQL injection de PortSwigger. Resolver primero de manera manual y luego usando sqlmap.

  4. Resolver los laboratorios de OS Command Injection de PortSwigger.

Web III#

  1. En Damn Vulnerable Web Application aprovecharse del RCE en las secciones de Command Injection y File Upload para obtener una shell reversa.
